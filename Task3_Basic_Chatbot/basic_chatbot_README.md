#  Basic Chatbot

# Overview

The **Basic Chatbot** is a command-line Python application developed as part of the **CodeAlpha Python Programming Internship**. It is a simple rule-based chatbot that interacts with users by responding to predefined greetings and questions.

The chatbot accepts user input, matches it with predefined responses, and continues the conversation until the user types an exit command. This project demonstrates the implementation of basic conversational logic using Python.

# Objective

The objective of this project is to build an interactive chatbot that allows users to:

- Communicate through a simple command-line interface.
- Receive responses to common greetings and questions.
- Continue the conversation until the user chooses to exit.
- Demonstrate the fundamentals of conversational programming.

# Features

-  Greets users with friendly messages.
-  Responds to predefined questions and greetings.
-  Supports continuous conversation.
-  Ends the conversation when the user types **bye**.
-  Simple and user-friendly command-line interface.
